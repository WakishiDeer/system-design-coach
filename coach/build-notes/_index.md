## Adding a note

Notes are created by the **build-to-learn-lab** skill when you finish (or pause) a build and choose to
keep it:

1. Work in a gitignored lab `coach/labs/<NN>-<slug>/` (real, runnable code lives there).
2. When the study note is worth keeping, **graduate** it: copy the lab's `note.md` to
   `coach/build-notes/<slug>.md` and add a row to this table.
3. The lab folder stays local / gitignored; this graduated note is the durable, shareable version.

> ラボ（gitignore）で作業 → 完成した `note.md` だけをここに卒業させて tracked に残します
> （`coach/problems/local/` → `coach/problems/` の promote と同じ発想）。
