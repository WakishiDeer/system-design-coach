# Inbox (scratch)

Drop loose, at-hand design drafts here while you think — a draw.io file, a quick Mermaid sketch,
half an API, rough notes. They don't have to be tidy.

When you're ready, ask the coach to **"file my draft"** and it will move the draft into the
active session's `design/` folder as the next version (`vN.md`) and remove the loose copy, so
`coach/` stays clean.

This folder is intentionally a staging area; nothing here is graded until it's filed into a
session.
